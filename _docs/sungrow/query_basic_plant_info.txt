Query Basic Plant Info
Post
/openapi/platform/getPowerStationDetail

Query basic plant info by plant ID. Supports batch queries.
1. Request parameters
Request parameters
Parameter type
Required
Parameter example
Parameter description
ps_ids
String
Yes
123
The plant ID, separate multiple values with commas (can query up to 20 IDs—either the ps_ids or sns must be provided; when both are passed, the ps_ids is queried; the sns is only used if the ps_ids query returns empty)
sns
String
No
A123***,A123***,A123***
Communication Device SN (separate multiple values with commas (you can query up to 20 SNs); the ps_ids is queried and the sns is only used if the ps_ids query returns empty)
is_get_ps_remarks
String
No
1
Get Plant Remarks?: (1: Yes; defaults to no if not passed)

2.3 Error code
Error code
Description
1
Successful
0
Query failed. The interface is abnormal.
4
ps_ids in the input parameters is empty. No plants found through SNS.
5
Plant quantity of the query or sns exceeded the max. limit.
6
No plant information based on ps_ids is found.
7
All plants in ps_ids are not authorized.

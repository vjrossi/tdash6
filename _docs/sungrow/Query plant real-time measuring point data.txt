Query plant real-time measuring point data
Post
/openapi/platform/getPowerStationRealTimeData

Query plant real-time measuring point data by plant ps_id. Note: 1. This endpoint always returns the smallest unit of data. For example, Wh for yield, W for power, A for current, and V for voltage. 2. The data refreshes about every 5 minutes. It is recommended that the endpoint call frequency is no less than 5 minutes.
